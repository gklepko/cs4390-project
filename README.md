# CS4390 Group Project

## Project Overview

This project is a simple chat application built using Java, TCP Sockets, and Threads. 

It allows multiple clients to connect to a server and chat with each other in real time.

## Features
- Multiple clients can connect to the server
- Each client chooses their username
- Join/Leave notifications
- Side Pannel with real time updates of connected users
- Simple logging system that saves server activity within the logs/ folder

## Steps to run
1. Clone repository
2. Start the server 
````
Server.java
````

3. Start number of Client instances wanted
````
Client.java
````

4. Input information
````
Input Username, Address as "localhost", Port as "1234"
````

5. Click "Connect" Button
